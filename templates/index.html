Sentiment Analysis API


This is a sentiment analysis API with endpoint : https://sentiment-analysis-api.herokuapp.com/sentiment

Make a post request to get the sentiment and pass the text as string or json format 


            import requests
            import json

            data = {'text' : 'I am cool'}

            res = requests.post('https://sentiment-analysis-api.herokuapp.com/sentiment', json = data)

            print(res.text)
        

            Output :

                Positive
